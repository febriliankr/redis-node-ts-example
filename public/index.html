Basic Setup for Typescript-Express Server

How to start:

  • Clone the repository
  • `npm i` to install dependencies
  • `npm run build`
  • `npm run start`

Actual App routes:

  • /api/v1/quizzes
  • /api/v1/quizzes/answers
  • /api/v1/questions

Available routes:

  • /api/v1
  • /api/v1/user
  • /api/v1/quizzes
  • /api/v1/questions
  • /api/v1/user-answers
